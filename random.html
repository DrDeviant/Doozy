Random Useful Commands

Whats here:

  • Grep (-irl)
  • Cat (find particular line)
  • Install downloaded packages (direct website download)


Grep

  • grep -irl <string>   - can look in directories/files etc

Useful cat command to find particular line in a file:

  • cat -n <file name> 
  • cat <filename> | sed ' <linenumber>!d '

Installing downloaded packages (straight from website):

  • apt-get install -f ./<pkg name>
