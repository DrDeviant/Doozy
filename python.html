Python

Creating Virtual Enviroment:

  • python3 -m venv <name your venv>
  • source ./<name of venv>
  • . ./<path to venv>/bin/activate

Requirements text:

  • pip install -r requirements.txt

Using Pydoc3:

  • found in your venv dir, cd lib, cd python3.*, cd site-packages
  • To use: pdoc3 <py file>
  • Output as html/pdf : pdoc3 -html <py file>
  • Access via browser: file:///<path>
