Google Hacking

Cache - cache:website = Displays version of site in google cache instead of current version

Link - link:website = Lists any web pages that contain links to queried site

Info - info:website = Give info about listed page

Site - {keyword}site:website = Restricts search to specified location

Allintitle - allintitle:keyword = Returns words with specified keyword in the title

AllinURL - allinurl:keyword = Returns results only specified in query URL
